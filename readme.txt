This is plugin for inserting Gallery Slider to your blog's posts and pages
== Description ==

Plugin allow you create and end slide shows and insert it in your blog's posts and pages.


== Installation ==

Upload the AdGallery Slider plugin to your blog

Set chmod 777 to folder "slides" for upload images for slide show

Create slide show

Add slides to slide show

Insert code for slide show to post or page
